# Taller3
El taller consiste en la implementación de estructuras de datos tipo Grafos y Árboles B+ para simular el funcionamiento de acceso a archivos en un computador. La idea es optimizar los tiempos de acceso a estos aplicando dichas estructuras de forma eficiente.
